# 2FACE CRM (PWA-ready)

Мобильное CRM-приложение для модельной студии.  
Стек: `index.html` + React CDN + Tailwind + localStorage.

## Что внутри

- Компактный список учеников
- Категории: Дети / Подростки / Women / Интенсив
- Группы Women
- Периоды (абонементы) с историей
- Оплата: оплачено/долг, сумма, даты
- Занятия по периодам (дата, предмет, посещение, комментарий)
- Редактирование предметов
- Сохранение данных в localStorage

## Локальный запуск

Просто откройте файл `index.html` в браузере.

## Публикация на GitHub Pages

Этот репозиторий уже содержит workflow для автодеплоя:
`.github/workflows/pages.yml`

После пуша в ветку `main`:

1. В GitHub откройте **Settings → Pages**.
2. В разделе **Build and deployment** выберите **Source: GitHub Actions**.
3. Дождитесь выполнения workflow **Deploy static site to GitHub Pages**.
4. Сервис будет доступен по ссылке:
   `https://<your-username>.github.io/<repo-name>/`

Открывайте именно **`index.html`** в корне (страница по умолчанию на GitHub Pages). Файл `index_ready.html` только перенаправляет на актуальную версию. После деплоя при отсутствии обновлений сделайте жёсткое обновление страницы (кэш браузера).

## Важно про данные

Данные хранятся в `localStorage` конкретного браузера/устройства.
Они не синхронизируются между разными устройствами автоматически.
